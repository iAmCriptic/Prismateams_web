Lösungsvorschläge
1. E-Mail-Konfiguration prüfen

Stellen Sie sicher, dass alle E-Mail-Einstellungen in der .env Datei korrekt sind:

MAIL_SERVER=smtp.gmail.com
MAIL_PORT=587
MAIL_USE_TLS=True
MAIL_USERNAME=ihre-email@gmail.com
MAIL_PASSWORD=ihr-app-passwort
MAIL_DEFAULT_SENDER=ihre-email@gmail.com
2. Gmail App-Passwort verwenden

Für Gmail müssen Sie ein App-Passwort verwenden, nicht Ihr normales Passwort:

  1. Gehen Sie zu Google Konto Sicherheit
  2. Aktivieren Sie die 2-Faktor-Authentifizierung
  3. Erstellen Sie ein App-Passwort für "Mail"
  4. Verwenden Sie dieses Passwort in der .env Datei
3. Firewall/Netzwerk prüfen

Stellen Sie sicher, dass ausgehende SMTP-Verbindungen (Port 587) nicht blockiert sind.

4. Alternative E-Mail-Anbieter

Falls Gmail nicht funktioniert, versuchen Sie andere Anbieter:

  • Outlook/Hotmail: smtp-mail.outlook.com, Port 587
  • Yahoo: smtp.mail.yahoo.com, Port 587
  • Eigener Server: Ihre SMTP-Server-Details
